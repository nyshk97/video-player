# Vid プライバシーポリシー

制定日: 2026-06-27

Vid は、ユーザーの写真ライブラリに保存されている動画を端末内で再生、削除、トリミング保存するためのアプリです。

## 収集する情報

Vid は、開発者が利用者を識別できる個人情報、利用状況、写真ライブラリ内の動画、クラッシュ情報、広告識別子を収集しません。

## 写真ライブラリへのアクセス

Vid は、以下の機能を提供するために写真ライブラリへのアクセス許可を求めます。

- 動画の一覧表示
- 動画の再生
- 動画の削除
- トリミングした動画の保存

写真ライブラリ内の動画は端末内で処理され、開発者のサーバーや第三者のサーバーへ送信されません。

iCloud 写真を利用している場合、動画の取得時に Apple の仕組みにより iCloud から端末へ動画がダウンロードされることがあります。この処理によって、動画が開発者へ送信されることはありません。

## 第三者サービス

Vid は、広告、アクセス解析、クラッシュ解析、トラッキング目的の第三者 SDK を使用していません。

## データの保存

Vid がトリミングした動画は、ユーザーの操作により写真ライブラリへ保存されます。書き出し時に作成される一時ファイルは、処理完了後に削除されます。

Vid で動画を削除した場合、削除処理は iOS の写真ライブラリ機能に従って行われます。
