# Vid

Vid は、iPhone の写真ライブラリに保存されている動画をすばやく見返すための iOS アプリです。

## 主な機能

- 写真ライブラリ内の動画一覧表示
- 動画の再生、一時停止、シーク
- 再生速度の変更
- 長押しによる一時的な 2 倍速再生
- 左右スワイプによる前後の動画への移動
- 動画の削除
- 範囲指定によるトリミング保存
- プレイヤー画面での縦向き、横向き切り替え

## 対応環境

- iPhone
- iOS 17.0 以降
- 日本語

## ビルド

このプロジェクトは XcodeGen を使って Xcode プロジェクトを生成します。

```sh
mise run gen
mise run build
```

詳細な確認手順は [VERIFY.md](VERIFY.md) を参照してください。
