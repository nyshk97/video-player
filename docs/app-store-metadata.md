## Description

Vid は、写真ライブラリに保存された動画をすばやく見返すためのシンプルな動画プレイヤーです。

一覧から動画を選び、再生、一時停止、シーク、再生速度の変更をスムーズに操作できます。左右スワイプで前後の動画へ移動できるため、撮影した動画を続けて確認したいときにも便利です。

主な機能:

- 写真ライブラリ内の動画一覧表示
- 再生、一時停止、シーク
- 再生速度の変更
- 長押し中の一時的な 2 倍速再生
- 左右スワイプで前後の動画へ移動
- 不要な動画の削除
- 範囲指定によるトリミング保存
- 縦向き、横向きの切り替え

Vid は動画を外部サーバーへ送信しません。端末内の写真ライブラリを使って、ローカルで動画を扱います。
